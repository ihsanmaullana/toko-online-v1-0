# IMSTORE

**IMSTORE** adalah sebuah toko online yang menyediakan berbagai pilihan smartphone berkualitas dengan harga terbaik. Website ini dirancang untuk memberikan pengalaman belanja yang mudah dan menyenangkan dengan antarmuka pengguna yang modern dan responsif.

## Fitur

- **Promosi Utama:** Menampilkan pesan selamat datang di bagian atas halaman dengan tautan untuk mengecek produk terbaru.
- **Navigasi:** Menyediakan navigasi yang jelas dengan akses cepat ke panduan pembelian dan tombol untuk menghubungi admin melalui WhatsApp.
- **Call to Action (CTA):** Menawarkan pencarian produk dengan cepat melalui form yang responsif dan tombol pencarian yang menarik.
- **Daftar Produk:** Menampilkan produk-produk unggulan dengan informasi harga, deskripsi singkat, dan tautan untuk melakukan pembelian melalui WhatsApp.
- **Footer:** Menampilkan informasi hak cipta yang sederhana di bagian bawah halaman.

## Struktur Kode

### 1. Meta Tags

Bagian ini berisi meta tags untuk SEO dan optimasi media sosial:

- **Open Graph (og):** Meta tags untuk meningkatkan keterlihatan konten di Facebook.
- **Twitter Cards:** Meta tags yang memastikan konten Anda terlihat optimal saat dibagikan di Twitter.

### 2. Pustaka CSS

Menggunakan Tailwind CSS yang diimpor melalui CDN untuk styling halaman. Tailwind CSS dipilih karena fleksibilitasnya dalam pembuatan desain yang responsif dan konsisten.

### 3. Promotion Bar

Sebuah banner promosi yang ditampilkan di bagian atas halaman untuk menyambut pengunjung dan mengarahkan mereka ke produk terbaru.

### 4. Navigation Bar

Navigasi utama situs dengan tautan ke halaman panduan pembelian dan tombol untuk menghubungi admin melalui WhatsApp. Navigasi ini juga mencakup branding IMSTORE dengan logo dan tagline.

### 5. Call to Action (CTA)

Sebuah section yang mengajak pengunjung untuk mencari produk yang mereka inginkan. Form pencarian ini didesain responsif dan mudah diakses dari berbagai perangkat.

### 6. Daftar Produk

Menampilkan beberapa produk terbaru dengan gambar, nama produk, harga, dan tautan untuk membeli langsung melalui WhatsApp. Setiap produk juga dilengkapi dengan label "New" untuk menunjukkan produk terbaru.

### 7. Footer

Footer yang sederhana menampilkan logo IMSTORE dan informasi hak cipta.

## Penggunaan

1. **Kunjungi Situs:** Anda dapat mengakses situs ini melalui [IMSTORE](https://toko-online-v1-0.vercel.app).
2. **Navigasi:** Gunakan navigasi untuk melihat panduan pembelian atau menghubungi admin.
3. **Cari Produk:** Gunakan form pencarian untuk menemukan produk yang Anda inginkan.
4. **Beli Produk:** Klik tombol "Beli Sekarang" pada produk yang ingin Anda beli untuk langsung menghubungi admin melalui WhatsApp.

## Teknologi yang Digunakan

- **HTML5:** Untuk struktur halaman web.
- **Tailwind CSS:** Untuk styling dan desain responsif.
- **JavaScript (CDN):** Penggunaan Tailwind CSS dari CDN.
- **Open Graph dan Twitter Cards:** Untuk optimasi konten media sosial.
